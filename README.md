# Sweet Shop Management System

## Tech Stack
- Backend: Spring Boot, Spring Security, JWT, PostgreSQL
- Frontend: React.js
- Database: PostgreSQL

## Features
- User Registration & Login (JWT Authentication)
- View available sweets
- Add sweets (Admin)
- Purchase sweets
- Secure REST APIs

## How to Run

### Backend
```bash
cd backend
mvn spring-boot:run
### Frontend
cd frontend
npm install
npm start
